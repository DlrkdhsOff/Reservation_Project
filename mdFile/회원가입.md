# 회원가입


## 일반 회원가입일 경우

요청 URL : POST http://localhost:8080/create/account

#### Request Body
```json
{
  "userId": "아이디",
  "password": "비밀번호",
  "name": "이름",
  "tel": "전화번호"
}
```
#### 성공
```json
{
  "result":  true,
  "message": "성공 메세지."
}
```
##### 실패
```json
{
  "result":  false,
  "message": "실패 메세지."
}
```
